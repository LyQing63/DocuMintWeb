## 项目介绍

DocMint是一个智能编辑器项目，大部分的web后端业务采用Java，OCR以及LLM2部署在Flask框架下。
该项目拥有基础的登录注册等功能，并内嵌LLM2模型以及OCR，使得在编辑的过程中可以使用图片输入以、ai补全、ai优化结构等功能
